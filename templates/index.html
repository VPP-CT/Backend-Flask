Flight API endpoint and parameters:

/flights?
Parameter Detail Format Example
budget Total budget in dollar for the whole trip Integer 2000
seg How many segments this trip has in total Integer 2
origin1 The first origin airport in this trip IATA airport code JFK
dest1 The first destination airport in this trip IATA airport code SFO
date1 The first departure date in this trip YYYY-MM-DD 2017-12-01
origin2 The second origin airport in this trip (optional) IATA airport code JFK
dest2 The second destination airport in this trip (optional) IATA airport code SFO
date2 The second departure date in this trip (optional) YYYY-MM-DD 2017-12-01

example query:

/flights?budget=2000&seg=1&origin1=nyc&dest1=sfo&date1=2017-10-31

/flights?budget=2000&seg=3&origin1=nyc&dest1=sfo&date1=2017-10-31&origin2=sfo&dest2=sea&date2=2017-11-25&origin3=sea&dest3=nrt&date3=2017-12-05

Hotel API endpoint and parameters:

/hotels?
Parameter Detail Format Example
city The city name of the target area City,Country OR City New%20york,USA OR New%20york
checkin The check in date for this stay YYYY-MM-DD 2017-12-01
checkout The check out date for this stay YYYY-MM-DD 2017-12-01

example query:

/hotels?city=new%20york&checkin=2017-11-05&checkout=2017-11-10
